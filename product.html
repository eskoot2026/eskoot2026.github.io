The Eskoot Module

On-scooter pipeline

A forward-facing camera streams frames locally to an edge compute board. The model detects hazards, computes directionality, and triggers alerts when risk exceeds a threshold. You can run fully offline, or sync events when connected.

Input
Camera + IMU
Compute
Edge board (Pi / Rock 5C Lite) + Custom MobileNetV3 AI Model
Output
Haptic trigger + event record

Haptic feedback system

Eskoot delivers non-intrusive, instant haptic alerts when a hazard is detected ahead. Alerts are intensity-controlled and adjusted with a distance-based fading curve helping riders react quickly without constant distractions.

Trigger logic
Confidence + distance + cooldown
Feedback modes
Pulse / ramp / burst patterns
Hardware
ERM/LRA motor + driver (handlebar / stem)
